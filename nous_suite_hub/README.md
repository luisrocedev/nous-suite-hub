# Nous Suite Hub

Proyecto de integración de módulos del catálogo Nous en suites empresariales de mayor impacto.

## Qué resuelve

En lugar de gestionar herramientas aisladas, permite agrupar módulos en suites con misión de negocio, estimar su ROI y registrar simulaciones de integración.

## Stack

- Flask + SQLite
- HTML, CSS, JavaScript

## Arranque

```bash
cd nous_suite_hub
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Abrir: http://127.0.0.1:5113
